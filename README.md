## 개요
메이플스토리 유저들을 위한 discord Bot 입니다. <br/>
간단한 사용자 정보와, 농장 정보, 이벤트 목록 등을 검색이 가능합니다.

기존에 사용하던 메봇? 이 작동을 안하게 되어 나도 한번 만들어 볼까 라는 생각으로 만들게 되었습니다.
